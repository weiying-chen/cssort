# CSSort

Sort CSS using piping

![](https://github.com/weiying-chen/cssort/blob/main/demo.webp)

## Installation

```bash
# Install dependencies
npm install
```

## Usage

```bash
# Run in the terminal
echo "display: 'block', color: 'red'" | ts-node script.ts

# Run in Helix
:pipe ts-node script.ts
```
